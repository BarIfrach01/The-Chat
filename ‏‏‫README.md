# Blazor WebAssembly Chat Application

פרויקט צ'אט מקוון הבנוי עם Blazor WebAssembly

## תכונות המערכת

### דף כניסה
- הרשמה של משתמש חדש
- התחברות למשתמשים קיימים
- מעבר לדף הצ'אט לאחר התחברות

### דף צ'אט אונליין
- צפייה בהודעות בסדר כרונולוגי
- רשימת משתמשים מחוברים
- הוספה, עריכה ומחיקה של הודעות
- עדכונים בזמן אמת
- סינון הודעות לפי מילות מפתח ותאריכים

### דף מנהל
- צפייה בנתוני Audit
- ניהול המערכת

## טכנולוגיות
- Blazor WebAssembly
- ASP.NET Core
- Entity Framework Core
- SignalR (לעדכונים בזמן אמת)
- Docker
